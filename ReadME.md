# DFYH引擎扩展
SilverFox Engine Expansion

DFYH引擎扩展是东方银狐一时兴起做的一个引擎扩展，基于天啟(Sovietianqi)的 [October-Revolution](https://github.com/Sovietianqi/October-Revolution) ，为 Yuri's Revenge 提供了一系列修复和辅助，以支持注入代码。
